# Rentment - Rental Platform (MERN Stack)

Rentment is a full-stack rental booking platform built using the MERN stack.

## Features
- User authentication
- Product listing & booking
- Card & UPI payment (Dummy gateway)
- Owner dashboard
- Admin panel

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## Setup Instructions

### Backend
cd backend
npm install
npm run server

### Client
cd client
npm install
npm run dev
